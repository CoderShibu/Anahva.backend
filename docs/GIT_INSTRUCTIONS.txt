It seems Git is not installed or configured in this terminal environment. 

Please install Git from https://git-scm.com/downloads if you haven't.

Once installed, open a terminal (Git Bash or Command Prompt) in this folder:
d:\Anahata

And run these commands one by one to push your code:

1. Initialize Git:
   git init

2. Add all files:
   git add .

3. Commit changes:
   git commit -m "Initial commit of Anahva - Privacy-first Wellness App"

4. Link your GitHub repository:
   git remote add origin https://github.com/CoderShibu/Anahva.git

5. Rename branch to main (standard practice):
   git branch -M main

6. Push to GitHub:
   git push -u origin main

(You may be asked to sign in to GitHub in the browser).
